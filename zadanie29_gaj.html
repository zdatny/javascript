1. W jaki sposób można określić właściwość danego obiektu?

Właściwość obiektu można określić na dwa sposoby:
auto.marka = "Opel";
auto["rok"] = 1996;

2. W jaki sposób można odwołać się do pól i metod?

Do pól i metod odwołujemy się za pomocą nazwy obiektu i kropki:
console.log(auto.marka);
auto.jedz();

3. Cechy charakterystyczne konstruktora

Konstruktor:

  • wywoływany jest automatycznie
  • nie zwraca żadnej wartości
  • może przyjmować argumenty
  • do tworzenia obiektu używa się słowa new

4. Przykład obiektu – trzy pola i jedna metoda

var osoba = {
imie: "Jan",
nazwisko: "Kowalski",
zawod: "piekarz",
wyswietl: function() {
document.write(this.imie + " " + this.nazwisko);
}
}

5. Znaczenie słowa this

Słowo this odnosi się do aktualnego obiektu i umożliwia odwołanie się do jego pól oraz metod.

6. Konstruktor w JavaScript

function Klient(nazwisko, imie, zawod) {
this.nazwisko = nazwisko;
this.imie = imie;
this.zawod = zawod;
}

7. Znaczenie słowa kluczowego new

Słowo new służy do tworzenia nowego obiektu na podstawie konstruktora:
var osoba1 = new Klient("Kowalski", "Jan", "kierowca");

8. Znaczenie słowa prototype

Prototype pozwala dodawać metody i właściwości wspólne dla wszystkich obiektów utworzonych na podstawie tego samego konstruktora.

Klient.prototype.wypisz = function() {
document.write(this.imie + " " + this.nazwisko);
}
