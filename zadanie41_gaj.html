pętla for

Konstrukcja: for(i=1;i<=10;i++)

Opis:< Pętla wykonuje się określoną liczbę razy.

Schemat blokowy: start → warunek → wykonaj instrukcje → zwiększ licznik

pętla while

Konstrukcja: while(warunek)

Opis: Najpierw sprawdzany jest warunek, potem wykonywane są instrukcje.

chemat blokowy: start → sprawdź warunek → wykonaj instrukcje

Pętla do while

Konstrukcja: do...while(warunek)

Opis: Instrukcja wykona się przynajmniej jeden raz, bo warunek sprawdzany jest na końcu.

Schemat blokowy: start → wykonaj instrukcje → sprawdź warunek
