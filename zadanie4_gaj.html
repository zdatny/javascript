Teoria JavaScript - Filip Gajewski

1) Kiedy powstał JavaScript (podaj datę)?

JavaScript został stworzony w 1995 roku przez Brendana Eicha.

2) Dlaczego jest to język skryptowy?

Jest to język skryptowy, ponieważ działa na poziomie przeglądarki i nie wymaga kompilacji.

3) Gdzie jest wykonywany JS (klient lub serwer)?

JavaScript jest wykonywany po stronie klienta (przeglądarki).

4) Dlaczego JS to bezpieczne narzędzie?

JS jest bezpieczny, ponieważ działa w sandboxie przeglądarki, ograniczając dostęp do zasobów systemu.

5) Czy jest to język obiektowy? Jeśli tak, to dlaczego?

Tak, JavaScript jest językiem obiektowym, ponieważ obsługuje obiekty i klasy.

6) Wymień zalety JS:

Zaleta 1: Wysoka kompatybilność z przeglądarkami.

Zaleta 2: Możliwość tworzenia dynamicznych stron.

Zaleta 3: Łatwość integracji z innymi technologiami.

7) Framework to:

Framework to struktura umożliwiająca szybki rozwój aplikacji.

8) Do aplikacji mobilnych można użyć Framework:

Tak, można użyć frameworków jak React Native lub Ionic.

9) Aplikacje responsywne to:

Aplikacje, które dostosowują się do różnych rozmiarów ekranu i urządzeń.

10) Przepisz linie kodu i wyjaśnij:

‹input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')"›

Ten kod tworzy przycisk otwierający nowe okno po kliknięciu.

11) Przepisz linie kodu i wyjaśnij:

‹window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200")›

Ten kod otwiera nowe okno o określonych parametrach, takich jak wysokość, szerokość i pozycja.

12)Opisz następującą instrukcję window.close()

metoda JavaScript, która zamyka bieżące okno lub kartę przeglądarki, ale tylko wtedy, gdy zostało ono otwarte skryptem (np. przez window.open()). W innych przypadkach przeglądarka zwykle blokuje jej działanie ze względów bezpieczeństwa
