1) Rodzaje okienek

W JavaScript występują trzy podstawowe rodzaje okien dialogowych:
• Okienko alert – wyświetla okienko dialogowe z informacją oraz przyciskiem OK.
• Okienko prompt – generuje okno, w którym użytkownik może wpisać wartość, zwracaną następnie do programu.
• Okienko confirm – służy do tego, aby użytkownik coś potwierdził (np. decyzję), stąd nazwa confirm.

2) Konwersji liczb

Wczytanie zmiennej z użyciem funkcji prompt() wykonywane jest jako ciąg znaków, dlatego konieczna jest konwersja na typ liczbowy, aby możliwe było wykonywanie działań arytmetycznych.
Do konwersji używa się:
• parseFloat(zmienna) – konwersja zmiennej do typu float (liczba zmiennoprzecinkowa),
• parseInt(zmienna) – konwersja zmiennej do typu int (liczbowego),
• toString(zmienna) – konwersja zmiennej do typu łańcuchowego (string).

3) Przy nadawaniu nazw zmiennym należy przestrzegać pewnych zasad (trzy zasady)

1. Nazwa zmiennej musi zaczynać się od litery, znaku "$" lub "_". Może zawierać tylko litery (bez polskich znaków), cyfry oraz te dwa znaki specjalne.
Poprawne nazwy: zmienna1, $wynik, _rozmiar, wynik1, Rekord, $_gracz.
Błędne nazwy: 1zmienna, wynik gry, wysokosc&szerokosc, wynik.rekord, &gracz, wysokość, imie+nazwisko.

2. Wielkość liter ma znaczenie – zmienne Rekord i rekord to dwie różne zmienne. To częsty błąd początkujących.

3. Nie należy używać słów kluczowych JavaScript jako nazw zmiennych.

4) Typy zmiennych w JavaScript

W JavaScript nie definiujemy typów zmiennych w momencie deklaracji, jak np. w C++. Typ jest określany w trakcie przypisywania wartości. Podstawowe typy to:
• Number – typ liczbowy (obejmuje liczby całkowite i zmiennoprzecinkowe),
• String – typ tekstowy (łańcuchy znaków),
• Boolean – typ logiczny (wartości: true lub false),
• Null – pusta zawartość,
• Undefined – typ niezidentyfikowany,
• Object – typ obiektowy.
